**Inception** is a Docker containerization and system administration project from the 42 School curriculum. It focuses on building a complete multi-service infrastructure using containers.

---

## Planned Overview

The project will set up a small-scale infrastructure composed of multiple Docker containers running services such as a web server, database, and application. Each service will be built from custom Dockerfiles and orchestrated together, emphasizing isolation, networking, and persistence.

## Planned Technologies

- Docker
- Virtualization
- System administration
- Container networking
- Service orchestration

## Current Status

This project is currently **in progress**. The source code and detailed implementation notes will be added once the project is completed.
