**ft_nmap** is a custom network scanner capable of detecting open ports and identifying services on target hosts. It is part of the advanced 42 School curriculum and focuses on low-level network programming.

---

## Planned Overview

The scanner will send probes to target hosts and analyze responses to determine which ports are open, closed, or filtered. It will support multiple scan types and service detection using raw sockets and direct protocol manipulation.

## Planned Technologies

- Network programming
- TCP/UDP sockets
- Raw sockets
- Protocol analysis

This project is currently **in progress**. The source code and detailed implementation notes will be added once the project is completed.
