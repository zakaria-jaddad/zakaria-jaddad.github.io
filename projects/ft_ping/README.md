**ft_ping** is an implementation of the classic ICMP echo request/reply utility used to test network connectivity. It is part of the advanced 42 School curriculum.

---

## Planned Overview

The program will craft ICMP echo request packets, send them to a target host, and listen for echo replies. It will display round-trip times, packet loss statistics, and other diagnostic information similar to the standard `ping` utility.

## Planned Technologies

- ICMP protocol
- Raw sockets
- System calls
- Network diagnostics

## Current Status

This project is currently **in progress**. The source code and detailed implementation notes will be added once the project is completed.
