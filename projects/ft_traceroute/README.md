**ft_traceroute** is a network diagnostic tool that traces the path of packets across an IP network hop by hop. It is part of the advanced 42 School curriculum.

---

## Planned Overview

The tool will send packets with increasing TTL values to a destination and record the intermediate routers that respond with ICMP time-exceeded messages. This reveals the route taken by packets through the network.

## Planned Technologies

- ICMP protocol
- Raw sockets
- IP TTL manipulation
- Network diagnostics
